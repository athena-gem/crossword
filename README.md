# crossword
crossword puzzles for English learners in Japan

# クロスワード生成サイト

このプログラムは日本の小中学生向けに作られたクロスワード生成ツールです。  
任意の単語リストを入力すると、自動でクロスワードを生成します。

## 利用方法
1. index.html をブラウザで開く
2. 左側に単語リストを入力（例：soccer サッカー）
3. 「生成」ボタンを押すとクロスワードが作られます
